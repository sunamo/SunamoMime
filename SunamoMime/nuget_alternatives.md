# NuGet Alternatives to SunamoMime

This document lists popular NuGet packages that provide similar functionality to SunamoMime.

## Overview

MIME type utilities

## Alternative Packages

### MimeKit
- **NuGet**: MimeKit
- **Purpose**: MIME message library
- **Key Features**: MIME parsing, headers, multipart

### MimeMapping
- **NuGet**: MimeMapping
- **Purpose**: File extension to MIME mapping
- **Key Features**: Extension lookup, content type detection

### HeyRed.Mime
- **NuGet**: HeyRed.Mime
- **Purpose**: MIME detection
- **Key Features**: Content-based type detection

### MimeTypesMap
- **NuGet**: MimeTypesMap
- **Purpose**: MIME type mappings
- **Key Features**: Comprehensive extension database

## Comparison Notes

MimeKit for MIME messages. MimeMapping for file type detection.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **MimeKit**: MIME message library
- **MimeMapping**: File extension to MIME mapping
- **HeyRed.Mime**: MIME detection
